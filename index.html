Langua.js

THE JAVASCRIPT LIBRARY TO EASILY TRANSLATE WEBSITES.

Implementing? Easy as pie

Start with the basics

Import Jquery, texts and Langua

					<script src="jquery.min.js"></script>
					<script type="text/javascript" src="texts.json"></script>
	 				<script type="text/javascript" src="langua.js"></script>

Add a class

As natural as it sounds, add the class "translatable" to the text you want to select as translatable

					<p class="translatable">Example</p>

Indentify your text

Add a unique identifier to your texts

					<p class="translatable" id="text1">Example</p>

Save your texts

Use the texts.json file to write all your texts in all the languages needed

					var texts = {
						    "EN":[
						            {
						                "text1": "Example"
						            }
						    ],
						    "ES":[
						            {
						                "text1" : "Ejemplo"
						            }
						    ]
						};

Translate your page!

Call the function changeLang(lang), where lang is the language code (ES,EN, etc)

					<li>
						<a onclick="changeLang('ES')" >
							ES
						</a>
					</li>

Translate only one field?

As we said, easy as pie, call changeFieldText with the object to modify and the language code

					<li>
						<a onclick="changeFieldText($('#text1'),'EN')" >
							Text to EN
						</a>
					</li>
